# matercare-homes

AI-powered eldercare platform with passive IoT monitoring, RAG knowledge base, and MCP integration.

## Overview

Matercare Homes is an AI-powered eldercare platform delivering passive IoT-based monitoring, a retrieval-augmented generation (RAG) knowledge base, and Model Context Protocol (MCP) integration for caregiver workflows. It is part of the TAURUS AI SaaS ecosystem.
